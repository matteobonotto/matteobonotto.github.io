# GitHub Pages personal site

## Publish
1. Create a new repository named `<your-username>.github.io`
2. Copy these files into the repository
3. Edit `_config.yml` with your real email and links
4. Commit and push

## Customize
- Replace `assets/img/profile.png` with your preferred photo
- Edit `index.md` for your bio and experience
- Add blog posts in `_posts/`
- Optional: point `cv_url` in `_config.yml` to a PDF
